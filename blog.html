Question-1: What are the different ways to select an element in the DOM?

  • getElementById() – Selects one element by its ID.
  • getElementsByClassName() – Selects all elements with a class name.
  • getElementsByName() – Selects elements by their name attribute.
  • querySelector() – Selects the first element that matches a CSS selector.

    • Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML – Gets or sets HTML code (includes tags).
  • innerText – Gets or sets only visible text (hides CSS-hidden text).
  • textContent – Gets or sets all text (including hidden), faster than innerText.

    • Question-3: What is event delegation in the DOM?

    Event delegation means adding one event listener to a parent element to handle events from its children. It's useful when you have many similar elements.

    Question-4: What is event bubbling in the DOM?

    Event bubbling is when an event starts at the clicked element and moves up through its parent elements to the top (document).

    Question-5: How do you create, add, and remove elements using JavaScript?

  • createElement() – Creates a new HTML element.
  • appendChild() – Adds the element at the end of a parent.
  • insertBefore() – Inserts before another element.
  • append() – Adds at the end (can add text or elements).
  • prepend() – Adds at the beginning.
  • replaceChild() – Replaces an old element with a new one.
